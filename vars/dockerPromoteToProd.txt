Runs the deployment pipeline that validates an approved Bitbucket Data Center pull request from devel to main and promotes the Docker image from dev Artifactory to prod Artifactory without modifying Git.

Example:

@Library('ci-shared-library') _
dockerPromoteToProd()

The values file can live outside the repository root. Configure its relative path with VALUES_PATH and configure the image fields with IMAGE_REPOSITORY_YQ_PATH and IMAGE_TAG_YQ_PATH.
