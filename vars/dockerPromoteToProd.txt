Runs the deployment pipeline that validates an approved Bitbucket Data Center pull request from devel to main, promotes the Docker image from dev Artifactory to prod Artifactory, and updates production Helm values.

Example:

@Library('ci-shared-library') _
dockerPromoteToProd()

The dev and prod values files can live outside the repository root. Configure their relative paths with VALUES_DEV_PATH and VALUES_PROD_PATH, and configure the shared image structure with IMAGE_REPOSITORY_YQ_PATH and IMAGE_TAG_YQ_PATH.
